Ferramenta de Anotação Semântica

Olá, especialista!

Seja bem-vindo(a) e muito obrigado por sua participação. Esta ferramenta é parte de uma pesquisa de mestrado que tem como objetivo validar um método para enriquecer semanticamente APIs do governo brasileiro.

Como funciona a atividade?

O processo é bem direto:

  1. Contexto: Você verá um campo de uma API (com sua descrição, tipo, exemplo, etc.).
  2. Sugestões: Você verá 3 sugestões de "tags" semânticas (do padrão Schema.org) pré-selecionadas para esse campo.
  3. Sua Tarefa: Sua missão é avaliar essas sugestões e escolher a tag que melhor descreve o campo.
  4. O Campo "Outra": Se nenhuma das 3 sugestões parecer correta, por favor, use o campo 'Outra' para digitar a tag que você, como especialista, julga ser a mais adequada. Se possível, tente usar um padrão de dicionário de dados (como o próprio Schema.org) para manter a consistência. Esta é uma das contribuições mais importantes!
Uso dos Dados

Os dados coletados aqui serão usados exclusivamente para fins acadêmicos. O objetivo é calcular a concordância entre os especialistas (IAA) e gerar um "dataset de ouro" (golden dataset) para treinar e avaliar modelos de IA.
